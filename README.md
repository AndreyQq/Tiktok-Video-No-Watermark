# Tiktok-Video-No-Watermark

Tiktok短视频去水印
### 在线测试：https://www.tikwm.com/

### 接口地址：https://www.tikwm.com/api/

### 请求方式：post

### 参数

plat - 平台（tik,snapchat,likee）

url - 短视频地址

### 返回结果：Json格式

{
"code": 0,
"msg": "success",
"data": {
"title": "",
"cover": "",
"play": "",
"wmplay": ""
}
}
